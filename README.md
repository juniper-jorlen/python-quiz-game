# Python Quiz Game

## Description

It is a simple command-line quiz game built with Python. The game asks the player multiple-choice questions, checks whether each answer is correct, keeps track of the score, and displays the final result at the end.

## Features

- Multiple-choice questions
- Instant feedback after each answer
- Score tracking
- Final score summary
- Easy to add more questions

## Technologies Used

- Python 3

## How to Run

1. Clone this repository:

```bash
git clone https://github.com/YOUR_USERNAME/python-quiz-game.git
```

2. Open the project folder:

```bash
cd python-quiz-game
```

3. Run the program:

```bash
python quiz.py
```
